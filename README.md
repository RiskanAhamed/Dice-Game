# 🎲 Ultimate Dice Battle Pro

A modern and interactive dice game built using **HTML, CSS, and JavaScript** featuring animations, score tracking, match history, and a Player vs Computer mode.

## 🌐 Live Demo

Play the game online:

👉 https://riskanahamed.github.io/Dice-Game/

## 🚀 Features

* 🎮 Player vs Computer gameplay
* 🎲 Animated dice rolling effects
* 🏆 First-to-10 winning system
* 📜 Match history tracking
* ✨ Confetti celebration on victory
* 🌙 Modern dark-themed UI
* 📱 Responsive design for desktop and mobile

## 🎯 How to Play

1. Open `index.html` in your browser.
2. Click the **Roll Dice** button.
3. Both Player and Computer roll a dice.
4. The higher number wins the round.
5. The winner earns 1 point.
6. The first player to reach **10 points** wins the match.
7. A confetti animation appears when the match ends.
